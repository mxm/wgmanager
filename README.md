WG Manager
=========

New version of the infamous WG Manager which helps communities (e.g. people living in shared flats) to keep track of their spendings.

Unfortunately, it remains a work in progress.

Requirements
------------

* Python 3.4
* Django 1.7
* Allauth 0.18.0
* django-sendmail-backend 0.1.2
* django-taggit 0.12.2
